Arethos — Mental Wellness App

React Native · Expo SDK 54 · SQLite · Groq & Anthropic AI

3-person team project — Co-built a cross-platform mental wellness app designed specifically for KNUST students, addressing real mental health gaps on campus. The app provides three core experiences: an AI-powered Journal for reflective writing, a Mood tracker with sleep and energy logging, and a History/Reflections tab that surfaces behavioral patterns in plain language.

Engineered a dual-AI fallback architecture (Groq primary, Anthropic secondary) to ensure 99%+ uptime for the reflection engine. Built a custom haptic feedback system with multi-pulse waveforms calibrated through 9 iterations on Android hardware — solving the limitation that expo-haptics only fires single-pulse vibrations on Android by dropping to React Native’s native Vibration.vibrate() API. Implemented a PanResponder-based sleep slider with ref-pinned state to eliminate stale-closure bugs inherent in React Native gesture handlers.

Other technical contributions include a Modal-based bottom-sheet delete system (bypassing React Native’s unreliable zIndex between non-sibling elements), voice recording with real-time waveform visualization, dark/light theme toggle, and a 16-chapter technical documentation covering every architectural decision. The team used SQLite for fully offline-first local storage — no user data ever leaves the device.
